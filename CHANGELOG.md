# Changelog

Все заметные изменения в OLCVPN документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект следует [семантическому версионированию](https://semver.org/lang/ru/) (SemVer).

## [1.2.2] - 2026-06-19
### Изменено
- Иконка приложения теперь генерируется автоматически при сборке: скрипт `Scripts/make_icon.py` (Pillow) рисует монохромную иконку — белое кольцо с бегущей дугой на почти-чёрном фоне — в `App/Assets.xcassets/AppIcon.appiconset/icon-1024.png` (вызывается из `build.sh`). Добавлять PNG вручную больше не нужно.

## [1.2.1] - 2026-06-19
### Исправлено
- Кнопка удаления при свайпе профиля больше не белая: раньше она наследовала глобальный белый акцент приложения и торчала белым блоком за краем экрана. Теперь у неё тёмный фон (`Theme.bgRaised`) в тон минималистичной теме.

## [1.2.0] - 2026-06-19
### Добавлено
- На экране подключения кнопку теперь можно нажать во время подключения — «Отменить» прерывает попытку соединения (раньше кнопка была заблокирована, пока идёт подключение).
- В списке профилей — удаление свайпом влево (full-swipe удаляет сразу).
### Исправлено
- Вращающееся кольцо на экране подключения больше не дёргается и не «замирает». Анимация `RotatingRing` переведена с `withAnimation(...).repeatForever(...)` на `TimelineView(.animation)`, поэтому вращение не сбивается при ре-рендере (смена статуса, соседние анимации) и возврате из фона.
### Изменено
- Новая иконка приложения в стиле минимализма (белое кольцо с бегущей дугой на почти-чёрном фоне).

## [1.1.0] - 2026-06-19
### Изменено
- Полный редизайн интерфейса: минималистичная чёрно-белая тема (Mono). Убраны цветные aurora-акценты, текст белый на почти-чёрном фоне.
- На экране подключения — анимированная «бегущая» линия по рамке круга (`RotatingRing`) вместо цветного орба. Скорость вращения растёт при подключении.
- Главная кнопка: белая заливка, чёрный текст.
### Удалено
- Вкладка «Прокси-режим» и экран `ProxyTestView`. Механизм SOCKS5 внутри туннеля сохранён.
### Безопасность / стабильность
- В extension туннеля принудительно используется лёгкий транспорт `datachannel`: тяжёлые видео-транспорты (`vp8channel`/`videochannel`/`seichannel`) пробивают лимит памяти Network Extension (~50 MB) и приводят к падению. На iOS не вызывается `SetProtector` (актуально только для Android).

## [1.0.1] - 2026-06-19
### Исправлено
- UI больше не выходит за края экрана на всех вкладках. `AuroraBackground` сделан нейтральным к компоновке (`GeometryReader` + фиксированный размер по экрану + `.clipped()`).

## [1.0.0] - 2026-06-19
### Добавлено
- Базовая версия OLCVPN: подключение, профили, прокси-режим, настройки.
- Начало ведения версий по семантике.
