# Changelog

Все заметные изменения в OLCVPN документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект следует [семантическому версионированию](https://semver.org/lang/ru/) (SemVer).

## [1.1.0] - 2026-06-19
### Изменено
- Полный редизайн интерфейса: минималистичная чёрно-белая тема (Mono). Убраны цветные aurora-акценты, текст белый на почти-чёрном фоне.
- На экране подключения — анимированная «бегущая» линия по рамке круга (`RotatingRing`) вместо цветного орба. Скорость вращения растёт при подключении.
- Главная кнопка: белая заливка, чёрный текст.
### Удалено
- Вкладка «Прокси-режим» и экран `ProxyTestView`. Механизм SOCKS5 внутри туннеля сохранён.
### Безопасность / стабильность
- В extension туннеля принудительно используется лёгкий транспорт `datachannel`: тяжёлые видео-транспорты (`vp8channel`/`videochannel`/`seichannel`) пробивают лимит памяти Network Extension (~50 MB) и приводят к падению. На iOS не вызывается `SetProtector` (актуально только для Android).

## [1.0.1] - 2026-06-19
### Исправлено
- UI больше не выходит за края экрана на всех вкладках. `AuroraBackground` сделан нейтральным к компоновке (`GeometryReader` + фиксированный размер по экрану + `.clipped()`). Раньше фиксированные «пятна» 460–520pt растягивали корневой `ZStack` каждого экрана до ~520pt, и контент срезался по бокам.

## [1.0.0] - 2026-06-19
### Добавлено
- Базовая версия OLCVPN: подключение, профили, прокси-режим, настройки.
- Начало ведения версий по семантике.
